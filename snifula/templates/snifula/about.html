About Snifula.com

Snifula.com is a forum site which was made as an experimental project and the current code for the website can be found on Github. Snifula was designed to be quite minimal but many features are sure to come as the project progresses.

Rules

Anything can be posted on this site by anybody as long as they have an account to post with; However, there are some limitations on what constitutes as 'anything'. The limitations are quite simple and anybody with common sense will understand them. Any violation of these rules will result in a permanent ban from the site.

  • No illegal content/activities
  • No spamming
  • No spreading of harmful websites

To re-iterate: anything goes as long as the content does not include or imply activities/content described above.
